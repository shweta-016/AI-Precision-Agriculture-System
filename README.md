# 🌱 AI Precision Agriculture System

An AI-powered agriculture decision support system that helps farmers make smarter farming decisions using machine learning and data analysis. The application provides crop recommendations, irrigation prediction, plant disease detection, weather information, and dataset insights through an interactive Streamlit dashboard.

⚙️ Tech Stack

Language: Python

Frontend / Dashboard: Streamlit

Machine Learning: Scikit-learn (Random Forest)

Deep Learning / Image Processing: TensorFlow, OpenCV

Data Processing: Pandas, NumPy

Model Storage: Joblib

API Integration: Weather API

Visualization: Streamlit Charts

This system demonstrates how AI and data science can be used to support precision agriculture and smart farming practices.
